Button Functions

The table shows the incoming data from the monitoring stations. Newly received data will add to the bottom of the table.

The Location button will sort the table data by the Locaiton the data came from. It will also add the data to the linked list.

The Vehicle# will sort the table data by Total Number of Vehicles. It will also add the data to the binary tree.

The Velocity will sort the table data by Average Velocity of the vehicles.

The information panel will display system messages.

The Check Status button will send a message to the monitoring stations to see which ones are still connected. A list of connected staions will be displayed.

The linked list section will display the sorted data from the linked list.

The binary tree section will display the data from the binary tree.

Enter a value in the Search box and click the Search button to determine if the value is in tree.

The Display button will display the data in the binary tree in a 2D graphical represntation.

The Order Display buttons will display the data in the binary tree in the relevant order.

The Save buttons will save the data in the binary tree to a file in the relevant order.

The Exit button will close all connections and close the application.
