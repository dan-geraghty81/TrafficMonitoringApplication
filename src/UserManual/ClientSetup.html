Monitoring Station Configuration

Setup and configure the Monitoring Station to connect with the central Monitoring Office.

Screen

Details

Select the IP address for the central monitoring office from the list provided or enter the IP address.

Enter the port number of the server in the port number field.

Enter the Monitoring Station number in the station number field.

Once all the fields have been completed click the Start Client button to connect the monitoring station to the monitoring office

Back to Top

Screen Layout

Screen

Details

Enter the time of the current reading in the time field.

The location number will be automatically populated based on the station number.

Enter the number of lanes for the section of road in the no of lanes field.

Enter the total number of vehicles recorded for that time period in the total no of vehicles field

The average number of vehicles will automatically populate after the number of lanes and total vehicles fields have been filled in.

Enter the average velocity of the vehicles in the average velocity field.

Once all fields have been filled in click the Submit button to send the information to the monitoring office.
